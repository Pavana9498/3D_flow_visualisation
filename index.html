Flow Visualization with Three.js

Using Flow Visualization

  • Move the cylinder using 'up' and 'down' arrows
  • Rotate & Zoom the point cloud cylinder using mouse
  • Move the plane using 'a'(right) and 'd'(left) keys to see intersecting points with point cloud on a scatter plot
  • Brushing and linking on scatter plot will be visible by selecting the points. You can see the concentration of the corresponding points.
  • The rest of the points other than the intersection points are greyed out.
